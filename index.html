群

定义 2.1 (群). 称集合\(G\)与运算\(\cdot : G\times G \rightarrow G\)为群, 当且仅当满足以下公理, \(\forall x, y\in G\)

  1. \(x(yz) = (xy)z\)

  2. \((\exists e\in G)\ ex = xe = x\)

  3. \((\exists a^{-1}\in G)\ aa^{-1}= a^{-1}a = e\)

若又满足\((\forall a, b \in G)\ ab = ba\), 则称为Abel群.

定义 2.2 (群直积). \(G, H\)群, 称\(G\times H\)为群直积, 即其为集合\(G\times H\)与运算\(((x, a), (y, b)) \mapsto (xy, ab)\).

语式和时态

直陈式

  1. 直陈式现在时
    1. er / ir / III
      1. e / es / e / ons / ez / ent
        1. 发音
        2. mange -mangeons
      2. is / is / it / issons / issez / issent
      3. être / avoir / aller
    2. 用法
      1. 正在发生的动作
      2. 描述当前状况
        Je suis content car ça marche (bien).
      3. 现在习惯重复
        Chaque week-end, toute ma famille va à la campagne en voiture.
      4. 离现在较近
        Je pars demains.
  2. 最近将来“时”，仅口语使用
    1. 短时间即将要做
    2. aller : 半助动词
      je vais aller au théater avec mes amis après l'école.
      1. ne ... pas : 变位动词两侧
        je ne vais pas aller au théater ...
        nous n'allons pas acheter cet appartement.
      2. 代词提前 : 相关动词（词义相关）
        nous allons l'acheter COD
  3. 过去最近将来“时”
    aller的未完成过去时 + 动词不定式
    1. 间接引语
      Il a dit : « Je vais aller au cinéma. » -> Il a dit qu'il allait aller au cinima.
    2. 时间状语从句
      Quand j'ai renconté Paul, il allait faire des courses.
  4. 最近过去“时”
    venir de + v.inf.
    elle vient de finir son travail.
    1. ne ... pas
      elle ne vient pas de finir son travail.
    2. 代词提前 : 相关动词（词义相关）
      elle vient de le finir.
  5. 过去最近过去“时”
    venir未完成过去时 + de + v.inf.
    1. Il a dit : « Je viens d'aller au cinéma. » -> Il a dit qu'il venait aller au cinima.
    2. Quand Léa est arrivé il venait de pleuvoir.
  6. 简单将来时
    直陈式现在时第一人称复数 / 特殊词头 + ai / as / a / ons / ez / ont
    时间状语
    1. 副词
    2. bientôt / prochainement
    3. 名词短语
          • prochain(e)
        1. le mois prochain / la semaine prochaine
    4. 介词短语
      1. dans + 段时间 : 在...之后
      2. dans un avenir proche. : 在不久的将来
        Tu vas réussir dans un avenir poche.
      3. dans le futur
      4. dans les + (num.) prochaines années : 在接下来的...内
  7. 复合过去式 过去某时间完成的，“了”，讲述
    avoir / être + p.p. : I : é / II : i / III
    1. 时态条件
      1. 具体时刻完成的
        J'ai perdu mon travail il y a six mois.
      2. 某时间段完成的动作
        Elles ont travaillé vingt ans dans cette entreprise.
    2. avoir 同前置直接宾语性数配合。
      1. COD
        j'ai vu la femme. -> je l'ai vue.
        en 不必配合
      2. que 引导的关系从句
        C'est un table que nous avons mangé
      3. 特殊疑问词+名词
        Quelle rode avez-vous acheteé ?
        Combien de rode avez-vous acheteé ?
    3. être 引导的不及物动词
      (de / re)venir / aller // arriver / partir // (r)entrer / sortir // retourner // monter / descender / tomber // naître / mourir / passer 的不及物词性
      必须性数配合
    4. être 引导的代动词
      1. 自反/相互
        1. se [COD] -> 性数配合
          Nous nous sommes promenés hier.
          Nous nous sommes téléphoné hier.
        2. laver / brosser / raser
          elle s'est lavée.
          elle s'est lavé les mains.
          les mains à elle
      2. 被动/绝对
        必须和主语性数配合
        Elle s'en est allée
  8. 未完成过去时
    第一人称复数 + ais / ais / ait / ions / iez / aient
    在过去起止点不明确中持续或重复的动作状态，“着”，描述背景
    1. 持续（那段时间内一直）
      1. Ce n'était pas un métier intéressant.
    2. 重复（那段时间内重复着）
    3. 描述背景
      1. L'entreprise avait de problems, alors ils nous ont licenciés.

    动词不定式(简单将来时词根) + 未完成过去时词尾 = 过去将来时形式

    1. 简单句
      委婉/感叹语气
      1. 请求
        1. Je voudrais un cafe.
      2. 建议

      3. 推测（条件）= peut-être ...

      4. 惊讶愤慨
        1. Comment ? Vous feriez cela ?
    2. 复合句

R 6500 WH.

这是一个在我看来非常好玩的问题, 也困扰过我一段时间. 现在我将学到看到想到的东西记下来, 希望能够帮到大家.

我们不妨默认大家有一些奇怪的代数基础(知道向量空间所在的范畴, 其上的同态, 在基上的分解. 行列式的展开运算. 有限维对偶向量空间, 张量积*)和一些有趣的几何学基础(切向量的定义, 切空间的定义. 在高维空间(的滤子基)下的极限的定义), 除此之外, 不默认大家有任何内容未知.

现有\(E \subset\mathbb{R}^m, F \subset\mathbb{R}^n\)是两个度量\(\mathbb{R}\)-向量空间1的子开空间, 分别用指标\(i, \nu\)进行标记.

定义函数\(f\in \mathcal F(E, F) : E \rightarrow F : x \mapsto f(x)\), 显然有若给出\(F\)上的一族基, 有\(f(x)\equiv f^\nu(x)e_\nu\).

步骤1, 定义可微函数与增量.

我们称函数\(f\)为一个可微函数, 当且仅当 \[(\forall x, x_0 \in E) f(x) = f(x_0) + L\mid_{x_0}(x - x_0) + \alpha(x - x_0)\] 其中\(L\mid_{x_0}\in \mathcal L(\mathbb{R}^m, \mathbb{R}^n) = \mathcal L(T_{x_0}\mathbb{R}^m, T_{f(x_0)} \mathbb{R}^n)\)是一个线性映射. 而\(\alpha \in \mathcal F(\mathbb{R}^m, \mathbb{R}^n)\)在\(x\)趋向于\(x_0\)时表现为关于\(x - x_0\)的高阶小量, 即\(o(x - x_0)\).

我们记\(\Delta f := f(x) - f(x_0), \Delta x := x - x_0\), 称为\(f\)和\(x\)在\(x_0\)处的增量, 则 \[(\forall x, x_0 \in E) \Delta f\mid_{(x, x_0)} = L\mid_{x_0}(\Delta x\mid_{(x, x_0)}) + \alpha(\Delta x\mid_{(x, x_0)})\] 显然有\(\Delta f \in T_{f(x_0)}\mathbb{R}^n, \Delta x \in T_{x_0}\mathbb{R}^m\).

步骤2, 定义切映射(导映射, 微分)与导函数.

我们称\(L\mid_{x_0}\)为函数\(f\)在\(x_0\)处的切映射(导映射, 微分), 它将在\(T_{x_0}\mathbb{R}^m\)这个切空间内的切向量映成\(T_{f(x_0)} \mathbb{R}^n\)内的切向量, 记作\(Df\mid_{x_0}\). 注意, 这是一个线性映射. 这个映射可以看成是原函数\(f\)在\(x_0\)处的最好线性拟合2. 一个在有限维向量空间范畴上定义的线性函数可以通过在基\(\mathcal{B}\)上的分解来研究, 即 \[L \equiv L^\nu_i e^i\otimes e_\nu\] 由于\(\mathcal{L}(E, F)\)构成\(\mathbb{R}-\)向量空间, 其同构于\(E \otimes F^*\), 其中的元素\(L\)即是一个双重线性形式\(L \in \hom(E\times \hom(F, \mathbb{R}), \mathbb{R}) : (x, y^*)\mapsto L(x, y^*) := L^\nu_i e^i(x)y^*(e_\nu) =^\star L^\nu_i x^iy^*_\nu\), \(\star\)步需要我们亲爱的正交基底.

按照上面的说法, 我们能够将其写成矩阵\([L]_{\mathcal{B}} := (L^\nu_i)_{\nu, i}\)的形式. 这是映射的雅可比矩阵, 我们记这个矩阵为\(f'(x_0)\), 称为\(f\)在\(x_0\)处的导数.

由于我们发现, 上文中一直在回避\(x_0\)的变化, 现在我们开始讨论\(x_0\)的作用. 若\(x_0\)发生变动, 我们有切映射和导数都发生变动, 主要考察导数, 我们发现此时的"导数"可以看成是一个函数带入固定的\(x_0\)的值后的矩阵, 因此定义导函数\(f' : E \rightarrow M(m, n) : x \rightarrow f'(x)\).

步骤3, 定义偏导数, 复合函数的导数, 反函数的导数.

实际上, 我们发现存在另一种方式可以很好的求出 \(Df^\nu_i\mid_{x_0}\), 即 \[\partial_i f^\nu\mid_{x_0} := Df^\nu_i\mid_{x_0} := {\lim\limits_{h\rightarrow 0}} \dfrac{f^\nu(x_0 + he_i) - f^\nu(x_0)}{h}\] 这是由于将基选定后我们能够非常平凡地计算除法, 贯彻这种思想, 我们定义偏导数\(\partial_if^\nu\mid_{x_0} := Df\mid_{x_0}(e_i\otimes e^\nu)\), 有\(Df\mid_{x_0} = \partial_if^\nu e^i\otimes e_\nu\).

番外1, 对微分算子的浅层讨论.

1, 微分算子\(D\mid_{x_0} \in \mathcal{F}(\mathcal{D}^1(U(x_0), \mathbb{R}^n), \mathbb{R}^m\otimes\mathbb{R}^{n*}) : f\mid_{U(x_0)} \mapsto Df\mid_{x_0}\)是一个线性泛函. 并且满足莱布尼兹率(等价于保持复合运算3), 即如果我们给出了\(F\)上的内积\({\langle x, y\rangle} := g(x, y) := \rho_{\nu\mu} x^\nu y^\mu\), 即一个双线性形式\(\rho \in (F\times F)^*\), 那么 \[D{\langle f, g\rangle} = {\langle f, Dg\rangle} + {\langle Df, g\rangle}\] 其中\({\langle f, Dg\rangle} := {\langle f, D_ig\rangle}e^i\).

分量表达即为 \[D_i{\langle f, g\rangle} = {\langle f, D_i g\rangle} + {\langle D_f, g\rangle}\] 因此微分算子确实不保持内积.

步骤4, 隐函数定理, 反函数定理, 微分同胚.
