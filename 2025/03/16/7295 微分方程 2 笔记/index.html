调和函数的船新方法

Aim: 证明 Harnack 不等式

Harnack 不等式: \(\sup_{\Omega'} u \leq C\inf_{\Omega'} u\) 对于正调和函数 \(u\) 及 \(\Omega' \sqsubset \Omega\).

  • 传统证明: 积分, 平均值不等式, 对梯度再平均值.
  • 现在证明 (Nilenberg 等): 用极值原理.

Slogan: 只要对球做就好. 原因: 由于极大极小值 (能取到! 考虑 \(\bar\Omega'\) ??) 之间连线可被球有限覆盖.

只要有 (对数梯度估计) \(\sup_{B_{1 / 2}} \frac{|\partial u|}{u} \leq C\) 就好, (用 Newton-Leibnitz 公式).

怎么从梯度内估计 (P29) 推导对数梯度估计? 丘成桐 1975.

定理 (Yau). 若 \(u > 0 \in C^3(\Omega)\) 调和, 且 \(B_1 \sqsubset \Omega\), 那么 \(\sup_{B_{1 / 2}} |\partial\log u| \leq C_n\). 证明.

取 \(v = \log u\), 那么 \(u = \exp v\), \(u_i = \exp v.v_i\), \(0 = \Delta u = \exp v (|\partial v|^2 + \Delta v)\), i.e. \(\Delta v = - |\partial v|^2\) (1).

取辅助函数 \(\varphi = (1 - |x|^2)^2 |\partial u|^2\), 由于 \(\varphi\) 在边界等于 \(0\), 且 \(\varphi \geq 0\), 那么 \(\partial\varphi(x_0) = 0\) (2), \(\Delta \varphi(x_0) \leq 0\) (3). 通过 (1), (2), (3) 证明要证的 \(|\partial v| \leq C_n\).

取 \(\eta := 1 - |x|^2\), \(\varphi := \eta^2 |\partial v|^2\), \(0 = \varphi_i = 2\eta\eta_i |\partial v|^2 + \eta^2 2 |\partial v||\partial v|_i\), i.e. \(-2\eta_i|\partial v|^2 = \eta (|\partial v|^2)_i\) (2').

再带入 (3): 即 \(0 \geq \Delta \varphi(x_0) = \Delta(\eta^2)|\partial v|^2 + 2(\eta^2)_i(|\partial v|^2)_i + \eta^2\Delta(|\partial v|^2)\).

  • 第一项: 低阶项, 留下.
  • 第二项: 用 (2') 处理, 如下
    • \(= -8\eta_i\eta_i |\partial v|^2\).
  • 第三项: 希望往 \(|\partial v|\) 方向化简. 具体计算如下
    • \(\Delta(|\partial v|^2) = (v_jv_j)_{ii} = 2v_{jii}v_j + 2v_{ji}v_{ji}\). 再用 \(v_{jii} = (\Delta v)_j\), 再用 (1), 得到 \(v_{jii} = - (|\partial v|^2)_j\).
    • 从而第三项 \(= 2\eta^2 (-|\partial v|^2_jv_j + v_{ij}v_{ij})\).
    • 由于 \(v_{ij}v_{ij} \geq \sum_i v_{ii}^2 \geq (\Delta v)^2 / n = |\partial v|^4 / n\). 最后是 Cauchy 不等式和 (1).
    • 从而大于 \(2\eta^2(|\partial v|^4 / n - |\partial v|_j^2v_j) = 2\eta^2|\partial v|^4 / n - 4\eta\eta_j|\partial v|^2v_j)\), 最后一步用了 (2').

于是 \(0 \geq \Delta(\eta^2)|\partial v|^2 -8\eta_i\eta_i |\partial v|^2 + 2\eta^2|\partial v|^4 / n - 4\eta\eta_j|\partial v|^2v_j)\). 同除 \(Bv\) (定义为 \(|\partial v|^2\)), 得到

\[0 \geq \Delta \eta^2 - 8B\eta + 2\eta^2 |\partial v|^2 / n - 4\eta\eta_jv_j.\]

用 Cauchy 不等式拆 \(4\eta\eta_i v_i\), 得到 \(\eta^2 Bv / n + 4n B\eta\), 带入 \(\eta_i = -2x_i\), \(B\eta = 4x^2\), \(\Delta \eta = -2n\), \(\Delta\eta^2 = 2\eta \Delta\eta + 4\eta_i\eta_i\). 于是带入, 并利用 \(\eta^2 < 1\), \(x^2 < 1\):

\[\begin{aligned}0 &\geq -4n\eta + 16x^2 - 32x^2 + \eta^2 Bv / n - 16nx^2\\ &\geq -4n - 16 -16n + \eta^2 Bv / n.\end{aligned}\]

即 \(20n + 16 \leq \eta^2 Bv / n\).

于是 \[\sup_{B_{1 / 2}} \eta^2 Bv / n \geq \sup_{B_{1 / 2}} Bv / n \inf_{B_{1 / 2}} \eta^2 \geq \frac{9}{16} \sup_{B_{1 / 2}} Bv / n.\]

于是再用 (1), 得到 \(\sup_{B_{1 / 2}} \Delta v \leq \frac{16}{9} n (20n + 16)\).

定理得证.
