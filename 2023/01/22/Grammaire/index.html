Schenat pas homot

2023-01-22
法语

Grammaire

这是对法语A1A2部分语法的整理.

语式和时态

直陈式

  1. 直陈式现在时
    1. er / ir / III
      1. e / es / e / ons / ez / ent
        1. 发音
        2. mange -mangeons
      2. is / is / it / issons / issez / issent
      3. être / avoir / aller
    2. 用法
      1. 正在发生的动作
      2. 描述当前状况
        Je suis content car ça marche (bien).
      3. 现在习惯重复
        Chaque week-end, toute ma famille va à la campagne en voiture.
      4. 离现在较近
        Je pars demains.
  2. 最近将来“时”，仅口语使用
    1. 短时间即将要做
    2. aller : 半助动词
      je vais aller au théater avec mes amis après l'école.
      1. ne ... pas : 变位动词两侧
        je ne vais pas aller au théater ...
        nous n'allons pas acheter cet appartement.
      2. 代词提前 : 相关动词（词义相关）
        nous allons l'acheter COD
  3. 过去最近将来“时”
    aller的未完成过去时 + 动词不定式
    1. 间接引语
      Il a dit : « Je vais aller au cinéma. » -> Il a dit qu'il allait aller au cinima.
    2. 时间状语从句
      Quand j'ai renconté Paul, il allait faire des courses.
  4. 最近过去“时”
    venir de + v.inf.
    elle vient de finir son travail.
    1. ne ... pas
      elle ne vient pas de finir son travail.
    2. 代词提前 : 相关动词（词义相关）
      elle vient de le finir.
  5. 过去最近过去“时”
    venir未完成过去时 + de + v.inf.
    1. Il a dit : « Je viens d'aller au cinéma. » -> Il a dit qu'il venait aller au cinima.
    2. Quand Léa est arrivé il venait de pleuvoir.
  6. 简单将来时
    直陈式现在时第一人称复数 / 特殊词头 + ai / as / a / ons / ez / ont
    时间状语
    1. 副词
    2. bientôt / prochainement
    3. 名词短语
          • prochain(e)
        1. le mois prochain / la semaine prochaine
    4. 介词短语
      1. dans + 段时间 : 在...之后
      2. dans un avenir proche. : 在不久的将来
        Tu vas réussir dans un avenir poche.
      3. dans le futur
      4. dans les + (num.) prochaines années : 在接下来的...内
  7. 复合过去式 过去某时间完成的，“了”，讲述
    avoir / être + p.p. : I : é / II : i / III
    1. 时态条件
      1. 具体时刻完成的
        J'ai perdu mon travail il y a six mois.
      2. 某时间段完成的动作
        Elles ont travaillé vingt ans dans cette entreprise.
    2. avoir 同前置直接宾语性数配合。
      1. COD
        j'ai vu la femme. -> je l'ai vue.
        en 不必配合
      2. que 引导的关系从句
        C'est un table que nous avons mangé
      3. 特殊疑问词+名词
        Quelle rode avez-vous acheteé ?
        Combien de rode avez-vous acheteé ?
    3. être 引导的不及物动词
      (de / re)venir / aller // arriver / partir // (r)entrer / sortir // retourner // monter / descender / tomber // naître / mourir / passer 的不及物词性
      必须性数配合
    4. être 引导的代动词
      1. 自反/相互
        1. se [COD] -> 性数配合
          Nous nous sommes promenés hier.
          Nous nous sommes téléphoné hier.
        2. laver / brosser / raser
          elle s'est lavée.
          elle s'est lavé les mains.
          les mains à elle
      2. 被动/绝对
        必须和主语性数配合
        Elle s'en est allée
  8. 未完成过去时
    第一人称复数 + ais / ais / ait / ions / iez / aient
    在过去起止点不明确中持续或重复的动作状态，“着”，描述背景
    1. 持续（那段时间内一直）
      1. Ce n'était pas un métier intéressant.
    2. 重复（那段时间内重复着）
    3. 描述背景
      1. L'entreprise avait de problems, alors ils nous ont licenciés.

条件式

  1. 条件式现在时
    动词不定式(简单将来时词根) + 未完成过去时词尾 = 过去将来时形式
    rais / rais / rait / riez / rions / raient
    1. 简单句
      委婉/感叹语气
      1. 请求
        1. Je voudrais un cafe.
      2. 建议
        1. Pourriez-vous venir un peu plus tôt ?
      3. 推测（条件）= peut-être ...
        1. Le directeur aurait beaucoup à dire sur cette affaire.
      4. 惊讶愤慨
        1. Comment ? Vous feriez cela ?
    2. 复合句
      si 引导条件状语 ：主将从现 / 未完成 + 条件（条件语气）
      1. 主将从现（真实）
        简将 / 近将 / 命令 / 直陈式现在时（口语）
      2. 条件语气
        没有发生，不能实现 / 现实相反
        前提虚假（如果我是你）/ 真实条件实现几率小
        1. Si je gagnais le gros lot, je ferais un tour du monde.

虚拟式

  1. 虚拟式现在时
    6 -ent + e / es / e / x / x / ent
    4 -ons + ions / iez
    avoir - ai / être - soi / aller - aill, all ...
    1. 用于que引导的补语从句
      表主观态度，是指情感主观，而非思维主观(penser / croire + 直陈)，但是否定同下
      1. 意愿
        aimer que(想要) / désirer / souhaiter / vouloir / demander / exiger / permettre -> devpasd
        espérer : 希望而非渴望，可以加将来（首选） / 条件
        主从句主语一致 -> 动词不定式
        1. Je veux que ma famille soit heureuse.
        2. Nous souhaitons qu'il réussisse à l'examen.
      2. 感情(sentiment)
        content / satisfait / heureux / peur / regretter / craindre / plaire / vexer
        1. Je regrette que ses parents ne le comprennent pas.
        2. Je suis content que vous alliez mieux.
        3. J'ai peur qu'il pleuve.
      3. 判断
        无人称句式
        il est ... que / il faut / il vaut mieux / il est temps / il semble / il suffit
        形容词为确定性陈述则直陈式，否定和疑问则虚拟，但是否定疑问则为直陈式
        主观态度
        1. Il est necessaire que nous apprenions bien le francais.
