基本定义

关于群, 子群, 同态, 商结构, 积结构.

群

定义 2.1 (群). 称集合\(G\)与运算\(\cdot : G\times G \rightarrow G\)为群, 当且仅当满足以下公理, \(\forall x, y\in G\)

  1. \(x(yz) = (xy)z\)

  2. \((\exists e\in G)\ ex = xe = x\)

  3. \((\exists a^{-1}\in G)\ aa^{-1}= a^{-1}a = e\)

若又满足\((\forall a, b \in G)\ ab = ba\), 则称为Abel群.

定义 2.2 (群直积). \(G, H\)群, 称\(G\times H\)为群直积, 即其为集合\(G\times H\)与运算\(((x, a), (y, b)) \mapsto (xy, ab)\).

注记 2.1. 若用加法运算, 则记为直和\(G \oplus H\).

定义 2.3 (中心). 称\(C \subset G\)为中心, 当且仅当\((\forall a\in G, c\in C)\ ac = ca\).

子群

考虑群\(G\).

定义 2.4 (子群). 集合\(H \subset G\)并且乘法\(\cdot\)在集合\(H\times H\)其上的限制封闭且满足群论公理的, 则\(H\)为\(G\)的子群, 记为\(H < G\).

命题 2.5 (子群判定). \(H < G \iff (\forall a, b\in H)\ ab^{-1}\in H\)

Proof. 参考相关书目. ◻

乘积定义

定义 2.6 (子群的乘法). \(H, K \subset G\), \(HK := \{hk \mid h\in H \land k\in K\}\)

注记 2.2. 子群的乘法不一定为子群.

命题 2.7. \(HK = KH \iff KH\)为子群.

定义 2.8 (元素与子群的乘法). \(\{a\}, H \subset G\), \(aH := \{ah \mid h \in H\}\)

定义 2.9 (子群的逆). \(A^{-1}:= \{a^{-1}\mid a\in A\}\).

命题 2.10 (子群的交仍是子群). \(H, K < G\), \(H \cap K (< G)\)称为子群的交.

Proof. 易证. ◻

生成

定义 2.11 (生成). \(X \subset G\), 称子群\(H\)为\(X\)生成的子群, 记作\(H = {\langle X\rangle}\), 当且仅当\(H\)是包含\(X\)的(关于\(\subset\)序的)最小子群, 换言之\(H = \bigcap_{K < G} K\).

定义 2.12 (单元素的生成). 单元素集合\(\{a\}\)的生成子群\({\langle\{a\}\rangle}\)为\(a\)的生成子群\({\langle a\rangle}\).

命题 2.13 (生成群的结构). \({\langle X\rangle} = \{\prod_i x_i^{\delta_i}\mid x_i\in X \land \delta_i = \pm 1 \land i\in I \land |I| < +\infty\}\)

推论 2.14 (单元素生成群 / 循环群). \({\langle a\rangle} = \{\ldots a^{-1}, 1, a, \ldots\}\)

命题 2.15. \(G = {\langle g\rangle} \land H = {\langle h\rangle} \implies(\forall x\in G\times H)\ |x| \mid \operatorname{l.c.m.}(|g|, |h|)\)

推论 2.16 (Abel群中的生成群). \({\langle X\rangle} = \{\prod_i x_i^{\lambda_i}\mid x_i \in X \land (i\neq j\rightarrow x_i\neq x_j) \land \lambda_i \in \mathbb{Z}\land i\in I \land |I| < +\infty\}\)

定义 2.17 (子群的并). 称子群\(L\)为子群\(H, K\)的并, 记作\(H\lor K\), 当且仅当\(L = {\langle H \cup K\rangle}\).

商群

陪集与正规子群

定义 2.18 (陪集). 称\(X (\subset G)\)为\(H (< G)\)在\(G\)中的一个(左)陪集, 当且仅当\((\exists a\in G)\ X = aH\).

注记 2.3.

  1. 陪集\(aH = \{b\mid a^{-1}b\in H\}\), 或者说\(b\in aH \iff a^{-1}b\in H\).

  2. \(|aH| = |H| = |Ha|\).

  3. \((\forall H < G)\ a\sim b: a^{-1}b\in H\)为等价关系, 对于\(a\)的等价类为\(aH\).

定义 2.19 (代表元系). 称\(A\)为子群\(H\)的(左)陪集代表元系, 当且仅当\((\forall a, b \in A)\ aH \neq bH \land \bigcup_{a\in A} aH = G\).

定义 2.20 (正规子群). 称\(H < G\)为正规子群, 记为\(H\lhd G\), 当且仅当\((\forall x \in H)\ xH = Hx\).

以下认为\(H < G \land K \lhd G\).

命题 2.21 (正规性).

  1. \(K \cap H \lhd H\)

  2. \(K \lhd K \lor H\)

  3. \(KH = K\lor H = HK\)

命题 2.22 (正规子群的判定). \(H\lhd G \iff aH = Ha \iff\)左陪集为右陪集\(\iff aNa^{-1}\subset N\). :::

商群

定义 2.23 (商群). 称集合\(\{aK\mid a\in G\}\)为\(G\)对\(K\)的商群, 记为\(G / K\), 其上乘法定义为\((aK, bK) \mapsto abK\).

Proof. 需证明良定义: \((\forall a, b \in G, \forall c, d \in G)\ aK = cK \land bK = dK \implies abK = cdK \iff (cd)^{-1}(ab) \in K \iff d^{-1}c^{-1}ab\in K \iff d^{-1}Kb \in K \iff Kb = dK \iff Kb = bK \iff K\lhd G\) ◻

同构

以下, 我们默认\(G, H\)为群, \(x, y, z \in G \land a, b, c \in H\).

定义 2.24 (同态). 称函数\(f : G \rightarrow H\)为群\(G\)到\(H\)的同态, 当且仅当\(\phi\)保持群运算, 即\(f(xy) = f(x)f(y)\).

推论 2.25. \(f(a^{-1}) = f(a)^{-1}\land f(e) = e\)

定义 2.26 (同构). 若同态\(f\)为函数单射则为单同态, 同理有满同态.

若同态\(f\)既单又满, 则为同构.

定义 2.27 (同态的核与像). 定义群论范畴上的同态\(f : G\rightarrow H\)的

  1. 核\(\ker f := \{x\mid f(x) = e\}\).

  2. 像\(\operatorname{Im}f := \{f(x)\mid x \in G\}\).

下面认为\(f : G \rightarrow H\)为同构.

命题 2.28 (核为正规子群, 像为子群). \(\ker f \lhd G \land \operatorname{Im}f < G\).

Proof. 对于\(\ker\)显然为子群, 对于正规性: \((\forall x\in G, k \in \ker f)\ f(xkx^{-1}) = f(x)f(k)f(x^{-1}) = 0 \implies xkx^{-1}\in \ker f \iff x\ker fx^{-1}\subset\ker f \iff \ker f \lhd G\), 由于.

对于\(\operatorname{Im}f\)有\((\forall a, b \in \operatorname{Im}f, \exists x, y \in G) f(x) = a, f(y) = b\), 从而\(ab = f(x)f(y) = f(xy) \in \operatorname{Im}f\). ◻

命题 2.29 (同构等价表述).

  1. 同构\(\iff\)有双边逆.

  2. 同构\(\iff \ker f = \{e\}\)

Proof. 显然. ◻

同构定理

命题 2.30 (正则射影与嵌入映射). \(K \lhd G, H < G\)

  1. \(\pi : G \rightarrow G / K : x \mapsto xK\)

  2. \(i : H \rightarrow G : x \mapsto x\).

推论 2.31. \(H \rightarrow HK / K\)存在自然的映射\(f = \pi\circ i\).

命题 2.32. \(K \lhd G \land K < \ker f \implies(\exists! \bar f)\ \bar f\circ\pi = f\)即

定理 2.33 (第一同构定理).

  1. \(G / \ker f \simeq \operatorname{Im}f\).

  3. \(f\implies(\exists! \bar f)\ i\circ\bar f\circ\pi = f\)

定理 2.34 (第二同构定理). \(H < G \land K \lhd G \implies H/(H\cap K) \simeq HK / K\).

定理 2.35 (第三同构定理). \(K < H \land K, H \lhd G \implies(G / K) / (H / K) \simeq G / H\)

定理 2.36 (像集). \(f: G \rightarrow H\)为满同态, \(S_f(G) := \{K\mid \ker f < K < G\}, S(H) := \{K\mid K < H\}\), 则 \(F : S_f(G) \rightarrow S(H) : K \mapsto f\{K\}\)为一一映射, 且将正规子群映成正规子群.

Proof. 显然有映射的良定义. 对于正规子群, 我们有\((\forall k \in K)\ f(xkx^{-1}) = f(x)f(k)f(x)^{-1}= yf(k)y^{-1}\implies(\forall y\in H)\ yf\{K\}y^{-1}\subset f\{K\}\) ◻

定理 2.37. \(N\lhd G \implies G / N\)中的(正规)子群形如\(K / N\), \(K\)为\(G\)中的(正规)子群 \(\land K \lhd G \iff K / N \iff G / N\).

直积与直和

在中我们定义了二元直积, 由于结合律我们可以推广到任意有限元直积, 现在我们需要定义无限元直积.

定义 2.38 (直积 / 完全直和). \(\{G_i\}_i\)是一族群, 他们的直积\(\prod_iG_i\)是它们的笛卡尔积与对应位置相乘作为乘法的群.

定义 2.39 (弱直积 / 直和). 直积\(\prod G\)的所有分量仅有有限个不为其对应子群的单位元的元素构成的子群\(H = \prod^{w} G\)称为弱直积.

命题 2.40 (有限元). 有限元情况下, 直积与弱直积相同.

以下认为\(G_i\)为一族群.

命题 2.41.

  1. \(\prod^w G \lhd \prod G\).

  2. 存在正则射影(满同态)\(\pi_i : \prod G \rightarrow G_i : f\mapsto f(i)\)与嵌入(单同态)\(i : G_i\rightarrow\prod^w G : x \mapsto x\delta(i)\).

推论 2.42. 嵌入将子群\(G_i\)映成\(\prod^w G\)中的正规子群.

定义 2.43 (映射的乘积). \(f_i : G_i \rightarrow H_i, f := \prod_i f_i : \prod G \rightarrow\prod H : a \rightarrow\{f_i(a_i)\}_i\).

定理 2.44. \(f(\prod^w G) \subset\prod^w H \land \ker f = \prod_i \ker f_i \land \operatorname{Im}f = \prod \operatorname{Im}f_i\).

推论 2.45. \(\prod G / \prod N \simeq \prod_i (G_i / N_i) \land \prod^w G / \prod^w N \simeq \prod^w_i (G_i / N_i)\)

泛性质

定理 2.46 (直积是群范畴的积). 若在群范畴内讨论问题, 以下说法等价.

  1. 直积是群范畴的积.

  2. \((\forall H, f_i : H \rightarrow G_i, \exists! f : H \rightarrow\prod G)\ f_i = \pi_i\circ f\).

定理 2.47 (直和是Abel群范畴的余积). 若在Abel群范畴内讨论问题, 以下说法等价.

  1. 直和是\(Abel\)群范畴的余积.

  2. \((\forall H, f_i : G_i \rightarrow H, \exists! f : \prod G \rightarrow H)\ f_i = f\circ i\)

内弱直积与剩余定理

定理 2.48 (同构). \(N_i\lhd G\)且

  1. \(G = {\langle N\rangle}\)

  2. \(N_i\cap{\langle N_j\rangle}_{j\neq i} = {\langle e\rangle}\), 则\(G\simeq \prod^w N\), 称为\(N_i\)的内弱直积.

定理 2.49 (元素形态). \(N_i\)是\(G\)中一族正规子群, 则\(G\)为其内弱直积\(\iff (\forall g\in G)\ g = \prod_in_{a_i}\), \(a_i\)为集合\(I = {1, \ldots n}\)的重排.

计数

以下认为\(G\)为群, \(K, H\)为\(G\)子群.

定义 2.50 (阶). 群\(G\)的阶数\(|G|\)为集合\(G\)的势.

定义 2.51 (元素的阶). \(a \in G\), \(a\)的阶\(|a|\)为群\({\langle a\rangle}\)的阶.

定义 2.52 (指数). 子群\(H\)在群\(G\)中的指数\(|G : H|\)为代表元系的势.

推论 2.53 (有限阶). 群\(G\)阶有限时有其子群有限阶, 且商集合\(G / H\)势有限, 其势为\(|G : H|\).

与同构定理的对应

同构定理要求的正规性全部可以利用指数去除. 以下认为\(K, H < G\)为群, 证明完全是写开利用陪集等势验证.

定理 2.54 (第二同阶定理). 在\(|H|, |K| < +\infty\)时, \(|H||K| / |H\cap K| = |HK|\).

推论 2.55 (与生成的组合). \(K, H\)有限阶则\(KH\)为有限阶.

定理 2.56 (第二同阶定理'). \(|H : H\cap K| = |HK : K|\).

推论 2.57. \(|H : H\cap K| \leq |G : K| \land\)等号在\(|G : K| <+\infty\land G \equiv HK\)时取得.

Proof. 考虑陪集\(H : H\cap K\)与\(G : K\), 构造映射\(f : H : H\cap K \rightarrow G : K : h(H\cap K)\mapsto hK\), 容易验证良定义及\(f\)为单射. 从而\(|H : H\cap K| \leq |G : K|\).

若有限情况取等, 则有\(f\)为满射, 则\(G = KH\). ◻

定理 2.58 (拉格朗日-第三同阶定理). \(K < H\) \(|G : H||H : K| = |G : K|\).

推论 2.59. 元素的阶是群阶的因数, 即\(|a| \mid |G|\).

推论 2.60. \(H, K\)指数有限, \(|G : H\cap K| \leq |G : H||H : K|\), 等号在\(G = HK\)时取得.

基本定义

环

定义 3.1 (环). Abel群\((R, +)\)与群上同态\(\cdot : R\times R \rightarrow R\)称为环.

注记 3.1.

  1. 若\(\cdot\)交换(是对称映射), 则为交换环.

  2. 若\(R\)中存在乘法单位元\(1\), 则称为含幺环.

定义 3.2 (零因子). \(x\in R^* :=\)左零因子\(\iff (\exists t\in R^*)\ tx = 0\).

注记 3.2. 同样定义右零因子, 零因子等. 若环无零因子, 则为无零因子环. 若无零因子环为Abel环, 则为整环.

定义 3.3 (可逆元). \(x\in R :=\)左可逆元\(\iff (\exists t\in R)\ tx = e\).

注记 3.3. 同样定义右可逆元, 可逆元等. 记可逆元集合为\(R^\times\). 若\(R^\times = R^*\), 则为体(除环). 若体为Abel环, 则为域. 此时\((R^*, \cdot)\)显然构成Abel群.

定义 3.4 (环直积). \(R, S\)群, 称\(R\times S\)为群直积, 即其为群\(R\oplus S\)与运算\(((x, a), (y, b)) \mapsto (xy, ab)\).

子环

定义 3.5 (子环). 子群\(S < R\)且对\(R\)中乘法封闭, 则为子环, 记为\(S < R\). \(R\)为\(S\)的扩环.

以下\(R, S\)为\(T\)的子环.

定义 3.6 (加法). \(R + S := \{r + s \mid r\in R\land s\in S\}\).

定义 3.7 (元素与子环的加法). \(r + S := \{r + s\mid s\in S\}\).

定义 3.8 (乘法). \(RS := \{rs\mid r\in R \land s\in S\}\), \((RS)\)为其生成的子环.

注记 3.4. 显然\(RS\)非子环.

命题 3.9 (子环的交). 子环的交仍是子环.

同构定理

命题 3.14 (正则射影与嵌入映射). \(I \lhd R, S < R\)

  1. \(\pi : R \rightarrow R / I : x \mapsto x + I\)

  2. \(i : S \rightarrow R : x \mapsto x\).

推论 3.15. \(S \rightarrow S + I / I\)存在自然的映射\(f = \pi\circ i\).

命题 3.16. \(K \lhd R \land K < \ker f \implies(\exists! \bar f)\ \bar f\circ\pi = f\)即

定理 3.17 (第一同构定理).

  3. \(f\implies(\exists! \bar f)\ i\circ\bar f\circ\pi = f\)
